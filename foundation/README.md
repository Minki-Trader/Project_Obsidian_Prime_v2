# Foundation

Shared reusable code and notes belong here.

- `config/` holds frozen shared configuration.
- `collectors/` holds source ingestion helpers.
- `features/` holds contract-aligned feature logic.
- `pipelines/` holds shared orchestration code.
- `mt5/` holds MT5-facing shared helpers.
- `parity/` holds Python to MT5 audit helpers and fixtures.
- `reports/` holds reusable shared notes.
